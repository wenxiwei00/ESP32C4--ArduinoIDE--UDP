# ESP32C4--ArduinoIDE--UDP
两个esp32c4之间互相传递数据。传递方式是udp。  

### 实现功能：  
转动我的esp32上的电位器，能控制别人esp32上的灯。彼此可以互相控制，因为数据通过udp传递。
